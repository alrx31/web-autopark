# web-autopark
The Autopark project

Stack:
server
  1) ASP.NET Core
  2) Dapper
  3) MS Server
  4) IRepository
  5) MVC
     
Client
  1) Razor Pages
  2) Bootstrap
  3) CSS3
  4) HTML5

IDE: JetBrains Rider (education only)

For start application run your MS Server and change connection string and run in terminal <code>dotnet run</code>
